⚡ Setup Installation Guide

Follow these five steps to get Tsunami Client up and running on your machine.
The process works the same on Windows, macOS, and Linux. No technical knowledge is needed. If you run into any issues, ask for help on the Discord server.

01

Install Fabric Loader

Head to the official Fabric website and download the Fabric Installer. Run the installer, select your Minecraft version (1.21-1.21.11 fabric), and click Install. The installer will create a new Fabric profile in your Minecraft launcher automatically.

02

Download Fabric API

Fabric API is a core library that provides common hooks used by Fabric mods. Download it from Modrinth or CurseForge. Make sure you pick the version that matches your Minecraft version. This file is required for Tsunami Client to work.

03

Download Tsunami Client

Grab the latest Tsunami Client JAR file from the download section above. Click "Select Version" and choose your Minecraft version. Always use this official website to ensure you get the authentic, unmodified version.

04

Move Files to Mods Folder

Place both JAR files (Fabric API and Tsunami Client) into your Minecraft mods folder. If the mods folder does not exist, create it manually.

🪟 Windows: %appdata%\.minecraft\mods
🍎 macOS: ~/Library/Application Support/minecraft/mods
🐧 Linux: ~/.minecraft/mods
05

Launch and Play

Open the Minecraft launcher and select the Fabric profile. Launch the game as usual. Once you are in-game, press Right Shift on your keyboard to open the Tsunami Client menu. From there you can enable, disable, and configure any module.
